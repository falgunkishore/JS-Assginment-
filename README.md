# JS-Assignment

Fundamentals of Web Design – Assignment 4
Language: JavaScript (using `prompt()` and `alert()`)

## Files

* q1.js – Digit Gatekeeper
* q2.js – Roll-Seed Lock
* q3.js – Mirror Corridor
* q4.js – Fare Calculator
* q5.js – Skipping Numbers
* q6.js – Contest Score Judge

## Approach

Each problem reads input using `prompt()`, processes the logic using loops/conditions, and displays the result using `alert()`.

## Time Complexity

* Q1: O((R − L) × digits)
* Q2: O(1)
* Q3: O(100000 × digits)
* Q4: O(1)
* Q5: O(m)
* Q6: O(1)

## Run

Open the `.js` file in a browser console and enter inputs when prompted.
